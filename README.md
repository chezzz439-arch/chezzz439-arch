I build real-world AI agents that solve actual problems — from predicting ER wait times to helping blind users navigate the world. I don't wait for permission to ship.

Most of what I build comes from a simple question: *why is this still broken?* ER Wait Oracle came from watching hospitals fail to manage surge capacity. AssistView came from realizing accessibility tools haven't kept up with modern AI.
